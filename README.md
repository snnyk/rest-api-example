# Example Rest API

Installation steps

```sh
# Clone the project
git clone https://github.com/snnyk/rest-api-example.git

# Jump to project folder
cd rest-api-example

# Install packages
composer install

# Get the project up
docker compose up

# Migrate database
php artisan migrate

# Seed data
php artisan db:seed

```
[Postman Api Doc. is here](https://documenter.getpostman.com/view/12113371/VUxKTUvT)
